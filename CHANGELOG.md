# Changelog

All notable public releases are recorded here.

## [0.1.39-beta] - 2026-07-14

### Added

- Native Android local-first learning workflow for Markdown notes, Capture,
  Review, search, backup, restore, and Trash.
- Optional OpenAI-compatible AI provider support protected with Android
  Keystore-backed local credential storage.
- Assistant Markdown rendering with GFM tables and confirmed Area-move actions.

### Security

- HTTPS-only provider endpoint validation and bounded backup import parsing.
- Assistant-proposed local changes require an explicit in-app action.

### Known limitations

- This is a GitHub-distributed Android beta, not a Play Store release.
- No account, desktop synchronization, or cloud backup is included.
