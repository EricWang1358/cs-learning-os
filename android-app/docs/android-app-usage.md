# CS Learning OS Android 使用说明

## 启动

安装 Android 包后打开 `CS Learning OS`。首次启动会导入内置学习内容；内容、阅读状态和复习记录保存在设备本地。

底部导航提供五个入口：Home、Capture、Library、Review 和 More。

## 首页

![Home 首页](images/android-home.png)

Home 是日常学习的起点：

1. 点击左上方的继续学习卡片，回到最近的知识节点。
2. 点击 `Capture`，快速记录尚未整理的想法或问题。
3. 点击 `Create`，创建新的 Markdown 知识节点。
4. 点击 `Search`，查找知识节点和题卡。

顶部的 `NODES` 和 `DUE` 分别显示本地知识节点数量与待复习数量。

## 知识库

![Library 知识库](images/android-library.png)

点击底部 `Library` 查看 Area 列表。每个 Area 显示节点数和已完成数；点击 Area 后可继续进入其中的节点。

常用操作：

1. 使用 `+ New area` 创建分类。
2. 在节点编辑器选择 Area 后保存，节点会出现在对应分类中。
3. 进入 `Trashbin` 查看已移入回收站的节点。

## 创建与编辑节点

从 Home 的 `Create` 或知识库中的分类进入编辑器。填写标题、选择 Area，并输入 Markdown 正文后保存。

保存失败时，不修改标题、正文或 Area 就重试，应用会复用同一个本地命令身份，避免生成重复节点。修改任一字段后会创建新的保存请求。保存成功、取消编辑、进入其他编辑器或切换到回收站时，旧请求身份会自动失效。

## Capture 与 Review

- `Capture` 用于先记录零散想法，再将其整理为节点。
- `Review` 显示待复习项目；完成复习后，待复习数量会更新。
- `More` 包含设置、本地数据与恢复相关入口。

## 数据与同步

当前版本以设备本地 Room 数据库为主。节点保存会原子写入节点、派生投影、幂等记录和本地 outbox；这保证重试与未来同步的基础一致性。

网络复制、电脑端同步和因果信封尚未实现。不要将本地 outbox 视为已经可用的跨设备同步功能。
