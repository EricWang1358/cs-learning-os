# Android Data Recovery

Backup schema v1 remains readable throughout the modular migration. Restore is an explicit, previewed operation; malformed or unsupported input must leave Room unchanged. Study Sync is not a backup replacement: it exchanges a phone-friendly study subset with the desktop, while JSON export/restore remains the complete phone recovery path.

Before any future destructive migration, export a backup through the in-app Backup screen and verify that the exported file can be selected for restore.
