# Demo Content

This is intentionally small but product-shaped. It proves that the app shell can ingest Markdown, serve nodes, search, open a detail page, group a mobile library by area/track/order, run more than one review card, and demonstrate Capture Slip workflows without private data.

Real notes, quiz banks, source traces, and `knowledge.db` belong in the ignored private `data/` directory or another path selected by `CS_LEARNING_DATA_ROOT`.
