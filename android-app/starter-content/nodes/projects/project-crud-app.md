# Project Pattern: CRUD

## Why It Matters

CRUD is the smallest useful full-stack project shape. It forces you to connect UI state, validation, routes, persistence, and error handling.

## Core Loop

Pick one record type, then implement create, read, update, and delete from the browser down to storage.
