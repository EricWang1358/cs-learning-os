# Capture Slip Workflow

## Phone Use Case

When typing a full note on a phone feels too heavy, capture only the smallest unclear point:

- the confusing sentence
- the topic hint
- the source label, such as lecture, video, or homework

## Later Processing

The slip can stay in the inbox, become a manual Markdown node, or be sent to a user-configured AI provider for an editable draft.

## Safety Rule

AI does not create a final node by itself. It opens Edit Mode, and the user must review the draft before tapping Save Markdown.
