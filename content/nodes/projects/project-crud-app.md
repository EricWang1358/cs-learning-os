# Project Pattern: CRUD

## Why It Matters

CRUD apps are small enough to finish and broad enough to expose routing, state, data modeling, validation, and persistence.

## Core Idea

Choose one domain object and implement the full lifecycle from UI interaction to database write and back.

## Practice / Application

- Todo tracker.
- Reading list.
- Course planner.
- Personal algorithm problem log.
