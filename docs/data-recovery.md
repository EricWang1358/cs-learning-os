# Data recovery

## English

Export a backup before restore, cleanup, migration, or changing phones. Keep a copy somewhere you can reach if the device is lost.

**Restore is a full replacement.** It deletes the app's current local notes and study state, then replaces them with the selected backup. Export the current device first if you may need anything on it later.

Trash is reversible: restore an item from Trash to return it to the Library. **Delete forever is only recoverable from backup.** The app has no undo after permanent deletion.

To recover:

1. Open More, then Data and Backup.
2. Select the backup file you trust.
3. Read the replacement warning and confirm only after exporting current data.
4. Check the Library and Review queue after restore.

## 中文

恢复、清理、迁移或换手机前，请先导出备份，并把副本保存在设备丢失后仍能访问的位置。

**恢复会完整替换数据。** App 会先删除当前本地笔记和学习状态，再用所选备份替换。如果以后可能还需要本机内容，请先导出。

回收站中的内容可以恢复到知识库。选择“永久删除”后，App 内无法撤销，只能从备份恢复。

恢复步骤：

1. 打开“更多”，进入“数据”和“备份”。
2. 选择可信的备份文件。
3. 阅读完整替换警告；确认当前数据已经导出后再继续。
4. 恢复后检查知识库和复习队列。
