# First run: five minutes

This walkthrough works in the Android beta and describes the same basic loop used on desktop. No account or AI setup is required.

## English

1. **Capture:** Open Capture, write one short idea or question, and save it.
2. **Library:** Open Library, choose an Area, and find or create a note.
3. **Review:** Open Review, reveal one answer, and rate how well you remembered it.
4. **Backup:** Open Backup and export a file before you restore, clean up, or move devices.

You can stop here. Notes, review cards, Trash, and backups work offline. AI is optional and only runs after you configure a provider.

## 中文

1. **捕捉：** 打开“捕捉”，写下一条简短想法或问题，然后保存。
2. **知识库：** 打开“知识库”，选择一个 Area，找到或新建笔记。
3. **复习：** 打开“复习”，显示一张卡片的答案，再评价记忆情况。
4. **备份：** 打开“备份”并导出文件；恢复、清理或换设备前都先导出。

到这里就完成了。笔记、复习卡、回收站和备份都能离线使用。AI 是可选功能，只有配置服务后才会运行。

For restore and deletion details, read [data-recovery.md](data-recovery.md).
